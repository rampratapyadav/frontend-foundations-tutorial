Intro to a Core Framework

Building with React & Vue

The Core Idea: Components

Both React and Vue are built around Component-Based Architecture.

Break down your UI into small, reusable, self-contained pieces.

React: Core Concepts

  • JSX: HTML-like syntax inside JavaScript.
  • Components: JavaScript functions that return JSX.
  • Props: How data is passed down to components (read-only).
  • State: A component's internal, changeable data (`useState` hook).

React Counter Example


import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}

Vue.js: Core Concepts

  • Single-File Components: Template, script, and style in one `.vue` file.
  • Template Syntax: Extends HTML with directives (`v-on`, `v-for`, etc.).
  • Reactivity: The `data()` function holds the component's state. When it changes, the template automatically updates.

Vue Counter Example


<template>
  <div>
    <p>You clicked {{ count }} times</p>
    <button @click="increment">Click me</button>
  </div>
</template>

<script>
export default {
  data() {
    return { count: 0 };
  },
  methods: {
    increment() {
      this.count++;
    }
  }
}
</script>

Key Takeaway

Frameworks help you build large, scalable applications using a component-based approach.
