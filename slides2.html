Today's Agenda

  • CSS Custom Properties (Variables) for Theming
  • Flexbox for 1D Layouts
  • CSS Grid for 2D Layouts

CSS Custom Properties

Define reusable values for consistent and maintainable styles.


:root {
  --primary-color: #007bff;
  --background-color: #ffffff;
}

body {
  background-color: var(--background-color);
}

[data-theme="dark"] {
  --primary-color: #bb86fc;
  --background-color: #121212;
}

Flexbox: 1D Layout

For arranging items in a single row or column. Perfect for headers and navigation.


header {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

CSS Grid: 2D Layout

For arranging items in both rows and columns. Perfect for page layouts and galleries.


.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 1.5rem;
}

Key Takeaway

Use Flexbox for 1D layout and Grid for 2D layout. Use Custom Properties for theming.
