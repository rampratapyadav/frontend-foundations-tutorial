Today's Agenda: 5 Use Cases

  1. Generating Boilerplate
  2. Writing Unit Tests
  3. Refactoring Code
  4. Debugging
  5. Explaining Complex Code

1. Generating Boilerplate

Save time on standard configuration files.


# Prompt Example:
"Generate a boilerplate `package.json` file for a new TypeScript project. 
Include scripts for `build` (running `tsc`) and `start` (running a local server)."

2. Writing Unit Tests

4. Debugging

Get help understanding and fixing cryptic errors.


# Prompt Example:
"I'm getting this error: `TypeError: Cannot read properties of undefined`. 
Here is the code that's causing it. Can you explain the likely cause and suggest a fix?"

// (Provide the problematic code block)

5. Explaining Complex Code

Understand complex algorithms or syntax quickly.


# Prompt Example:
"Explain what this regular expression does: 
/^\w+([\.-]?\w+)*@\w+([\.-]?\w+)*(\.\w{2,3})+$/"

Key Takeaway

AI is a tool to augment your skills, not replace them. Use it to automate the tedious parts of your job.
