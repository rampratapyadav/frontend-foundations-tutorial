What is the DOM?

The Document Object Model is a tree-like structure of your HTML, created by the browser. JavaScript can access and manipulate this tree.

1. Selecting Elements

Before you can manipulate an element, you have to select it.


// Get an element by its unique ID
const themeSwitcher = document.getElementById('theme-switcher');

// Get the first element that matches a CSS selector
const container = document.querySelector('#projects-container');

2. Handling Events

We use `addEventListener` to run a function when a user does something (like clicking).


const button = document.getElementById('my-button');

button.addEventListener('click', () => {
  // This code runs when the button is clicked
  console.log('Button was clicked!');
});

3. Manipulating Elements

Once selected, you can change an element's attributes, content, or style.


// Change an attribute
document.documentElement.setAttribute('data-theme', 'dark');

// Change the text content
status.textContent = 'Theme changed to dark.';

4. Creating & Adding Elements

We can create new elements from scratch and add them to the page.


// 1. Create a new element
const card = document.createElement('article');

// 2. Set its properties
card.className = 'project-card';
card.innerHTML = `<h3>New Project</h3>`;

// 3. Append it to a container on the page
const container = document.getElementById('projects-container');
container.appendChild(card);

Key Takeaway

JavaScript brings pages to life by selecting elements, responding to events, and manipulating the DOM.
